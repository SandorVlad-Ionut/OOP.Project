
The project's name is Plants and Flower Arrangements.
This program records every item in a flower shop. 
Product is the main class (which has price, height, name, type as attributes). It has 2 subclasses: Plant and FlowerArrangements.
While Plant has the pot type as attribute, FlowerArrangements has colour and use.
Products are saved in repository and an Id is generated for each new item.

Features:
-add a product
-delete a product
-show all plants or flowerarrangements
-show all the products in a price range
-display all products of a specific type
